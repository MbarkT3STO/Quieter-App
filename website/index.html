Features

Everything you need,
nothing you don't

Quieter is focused, transparent, and safe by design.

Instant performance gains

Disable CPU-hungry daemons like mediaanalysisd, chronod, and duetexpertd that are known to spike to 100–400% CPU on macOS Sonoma and Sequoia.

mediaanalysisd chronod duetexpertd triald

Always reversible

A full snapshot is written before every apply. One click in Settings restores everything to its original state.

Risk-aware warnings

Every service is rated Safe, Moderate, or Advanced. Advanced services require explicit confirmation before applying.

Live system stats

Real-time CPU and RAM monitoring with a 90-second sparkline chart. See the impact of your changes immediately.

Instant search

Filter across all 60 services by name, description, or category in real time. Find what you need in milliseconds.

Safety Model

Nothing changes
until you say so

Quieter is built around a strict safety-first philosophy.

01

Toggle = pending only

Flipping a toggle marks a service as pending. Nothing touches the system until you explicitly click Apply Changes.

02

Snapshot before apply

A full state backup is written to ~/.quieter/backup.json before any change is made. You can always restore it.

03

Rollback on failure

If any command fails mid-apply, all already-applied changes are automatically reversed. You never end up in a partial state.

04

Risk confirmation

Services marked Advanced require an explicit confirmation dialog listing exactly what you're about to change.
